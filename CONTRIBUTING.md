# Contributing

This repository is primarily an engineering documentation archive.

Contributions are welcome if they improve:

- Documentation accuracy
- Attribution correctness
- Build notes
- Hardware integration details
- Schematics clarification
- DSP explanation
- Historical context

Please respect the distinction between:

- The original Jeannie synthesizer by Rolf Degen / TubeOhm Instruments
- Third-party DSP libraries and examples
- RTAL integration and documentation work

Do not submit changes that remove or obscure original authorship or license information.
